Cloud Transcode

Distributed transcoding stack using Amazon AWS and FFMpeg. Free and Open Source.

Get Started! » Full doc »

Control Cost

  • You deploy the stack.
  • You control the resources you allocate to it.
  • You control speed and cost.

Scalable

The stack relies heavily on cloud services:

  • AWS SWF: Define your transcoding workflow dynamically and run the Activity you want in the order you want.
  • AWS SQS: Send JSON input through SQS. Receive updates from SQS. Simple, asynchonous, resilient and decoupled communication with the stack.
  • AWS Ec2: Add more machines == More horsepower. Transcode fast in the cloud. Spawn up to 1000 workers to transcode your jobs.
